# Политики

Официальные правила и регламенты работы в компании.

## Статьи в этом разделе

> Этот раздел пока пуст. Добавьте первую статью, создав файл `.md` в папке `docs/policies/`
> и добавив его в `nav:` блок файла `mkdocs.yml`.

---

!!! tip "Как добавить статью?"
    1. Перейдите в папку `docs/policies/` на GitHub
    2. Нажмите **Add file → Create new file**
    3. Назовите файл (например, `hr-policy.md`)
    4. Напишите содержимое и нажмите **Commit changes**
    5. Добавьте файл в `mkdocs.yml` в блок `nav:`
