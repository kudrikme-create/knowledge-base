# Процессы

Пошаговые описания рабочих процессов — как что делается в нашей команде.

## Статьи в этом разделе

> Этот раздел пока пуст. Добавьте первую статью, создав файл `.md` в папке `docs/processes/`
> и добавив его в `nav:` блок файла `mkdocs.yml`.

---

!!! tip "Как добавить статью?"
    1. Перейдите в папку `docs/processes/` на GitHub
    2. Нажмите **Add file → Create new file**
    3. Назовите файл (например, `sales-process.md`)
    4. Напишите содержимое и нажмите **Commit changes**
    5. Добавьте файл в `mkdocs.yml` в блок `nav:`
