# FAQ — Часто задаваемые вопросы

Ответы на самые распространённые вопросы команды.

---

## Как найти нужную информацию?

Используйте **поиск** (кнопка в правом верхнем углу или нажмите `/`) — он ищет по всему содержимому базы знаний.

## Как исправить ошибку в статье?

Нажмите кнопку **"Редактировать эту страницу"** в правом верхнем углу любой страницы. Вы перейдёте на GitHub, где можно отредактировать текст и сохранить изменения. Сайт обновится автоматически в течение ~90 секунд.

## Как добавить новую статью?

1. Войдите в GitHub репозиторий
2. Перейдите в нужную папку (`docs/guides/`, `docs/policies/` и т.д.)
3. Нажмите **Add file → Create new file**
4. Назовите файл с расширением `.md`
5. Напишите содержимое и нажмите **Commit changes**
6. Попросите ответственного добавить страницу в навигацию (`mkdocs.yml`)

## Чего нет в базе знаний?

Если вам нужна информация, которой нет в базе, напишите ответственному за базу знаний. Любая актуальная информация должна быть задокументирована здесь.
