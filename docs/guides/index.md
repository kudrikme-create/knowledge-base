# Гайды

Практические инструкции по инструментам, сервисам и повседневным задачам.

## Статьи в этом разделе

> Этот раздел пока пуст. Добавьте первую статью, создав файл `.md` в папке `docs/guides/`
> и добавив его в `nav:` блок файла `mkdocs.yml`.

---

!!! tip "Как добавить статью?"
    1. Перейдите в папку `docs/guides/` на GitHub
    2. Нажмите **Add file → Create new file**
    3. Назовите файл (например, `how-to-use-crm.md`)
    4. Напишите содержимое и нажмите **Commit changes**
    5. Добавьте файл в `mkdocs.yml` в блок `nav:`
